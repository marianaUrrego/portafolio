Vite + TypeScript + Sass**.

Este proyecto servirá como carta de presentación profesional para un
perfil de:

## 🛠 Instalación

### 1️⃣ Clonar repositorio

``` bash
git clone https://github.com/tu-usuario/portafolio.git
cd portafolio
```

### 2️⃣ Instalar dependencias

``` bash
npm install
```

### 3️⃣ Ejecutar en desarrollo

``` bash
npm run dev
```

Aplicación disponible en:

    http://localhost:5173

------------------------------------------------------------------------

## 🏗 Build de Producción

``` bash
npm run build
```

Genera carpeta `dist/` lista para despliegue.
